## Last Updated: 2025-12-21

## THE FUNDAMENTAL CLAIM

**HIRM proposes that consciousness emerges through self-reference phase transitions when information processing capacity C(t) exceeds a critical threshold.**

```
C(t) = Φ(t) × R(t) × D(t) ≥ C_critical ≈ 8.3 bits → Consciousness
```

This is FALSIFIABLE: If C(t) > 8.3 bits in clearly unconscious systems, HIRM is wrong.

**Claim:** Anesthesia systematically reduces C(t)
**Status:** UNTESTED

**Status:** UNTESTED

## THE ZERO-MULTIPLIER THEOREM

**Key HIRM insight:** Because C(t) = Φ × R × D, if ANY component = 0, then C(t) = 0

This explains:
- **Ant colonies:** High Φ (integration), but R = 0 (no self-reference) → Unconscious
- **Cerebellum:** High integration, but R ≈ 0 → No cerebellar consciousness
- **AI systems (current):** High Φ, high D, but R = 0 → Not conscious (yet)

**Falsification:** Find system with verified R = 0 that is conscious
