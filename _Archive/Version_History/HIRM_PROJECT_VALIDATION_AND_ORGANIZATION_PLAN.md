## VALIDATION METHODOLOGY

### How to Validate Content Preservation

**Tool 1: Text Comparison**
```bash
# Extract unique sentences from source
grep -v '^#' source_file.md | sort | uniq > source_unique.txt

# Extract sentences from consolidated
grep -v '^#' consolidated_file.md | sort | uniq > consolidated_unique.txt

# Find sentences in source but not in consolidated
comm -23 source_unique.txt consolidated_unique.txt > missing_content.txt

# Review missing_content.txt for substantive losses
```

**Tool 2: Citation Check**
```bash
# Extract all citations from source
grep -oE '\(([A-Z][a-z]+ et al\.|[A-Z][a-z]+( and [A-Z][a-z]+)?), [0-9]{4}\)' source_file.md | sort | uniq > source_citations.txt

# Extract all citations from consolidated
grep -oE '\(([A-Z][a-z]+ et al\.|[A-Z][a-z]+( and [A-Z][a-z]+)?), [0-9]{4}\)' consolidated_file.md | sort | uniq > consolidated_citations.txt

# Find citations in source but not in consolidated
comm -23 source_citations.txt consolidated_citations.txt > missing_citations.txt
```

